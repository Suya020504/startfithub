# StartFit QR Evidence Data

이 폴더는 QR 부록 사이트에 사용한 진단 과정과 대시보드 기능 화면을 스크린샷, 설명표, 보고서로 정리한 자료입니다.

## 최신 업데이트

- 최신 MVP의 `지원사업 추천` 상단 탭을 추가 반영했습니다.
- 대시보드 기능 화면을 최신 상단 탭 기준으로 다시 캡처했습니다.
- 대시보드 기능 정리 수량은 20개 화면입니다.

## 폴더 구성

| 폴더/파일 | 내용 |
|---|---|
| `diagnosis-process/` | 진단 과정 28개 화면 정리 |
| `dashboard-features/` | 대시보드 기능 20개 화면 정리 |
| `APPENDIX_CONTENT_REPORT.md` | 어떤 화면을 왜 정리했는지 설명하는 부록 기반 보고서 |

## 빠른 확인

- 진단 과정 표: `diagnosis-process/diagnosis-process-index.csv`
- 대시보드 기능 표: `dashboard-features/dashboard-features-index.csv`
- 보고서: `APPENDIX_CONTENT_REPORT.md`
