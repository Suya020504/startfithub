# Local Preview

This repository is the QR evidence hub site:

- Production: https://startfit-qr-evidence-hub.vercel.app/
- Short URL: https://startfithub.vercel.app/

Run the local preview from this exact folder:

```bash
npm start
```

Then open:

```text
http://localhost:4173
```

On Windows, you can also double-click:

```text
start-local.cmd
```

If another MVP screen opens, the local server was started from a different folder. Stop that server and run the command again inside this `startfithub` folder.
