# StartFit QR Evidence Hub

FIN:NECT StartFit 발표에서 QR로 함께 열어볼 수 있는 보조 자료 허브입니다.

발표를 대체하는 페이지가 아니라, 발표 중 다 보여주기 어려운 최신 MVP 화면, 진단 흐름, 데이터 기준, 검증 결과, 개인정보/신뢰 장치를 보완 설명하는 용도입니다.

## 핵심 메시지

StartFit은 예비창업자의 아이디어와 사업계획 데이터를 바탕으로 창업 준비도, 정책금융 준비도, 기관 연결 가능성을 진단합니다. 이 허브는 그 과정을 실제 화면과 근거 중심으로 보여줍니다.

## 화면 구성

| 구역 | 역할 |
|---|---|
| 첫 화면 | StartFit 보조 자료 허브 소개와 최신 대시보드 화면 |
| 개요 | 진단 점수 구조, 5대 준비도, 진단 흐름, 신뢰와 안전 |
| 진단 과정 | AI 진단 입력부터 대시보드 복귀까지 28개 화면을 큰 탭/세부 탭으로 정리 |
| 대시보드 기능 | 대시보드, AI 진단, 기관 추천, 지원사업 추천, 상담 관리, 성장 추적, 마이페이지 |
| 기술 영역 | AI 진단, 지원사업 추천 RAG, 기관 추천 스코어링, 공개/비공개 범위를 정보 제공 탭으로 정리 |
| 진단 결과 예시 | 핀넥트: 스타트링크 팀 데이터 예시본, 실제 우리팀 팀장의 A창업 입주기관 서류탈락 사례, B창업 입주기관 서류합격 사례를 통한 AI 정상작동 검증 |
| 데이터 기준 | 사용자 입력, 공공데이터, PSST 기반 점수화 근거, 공개/비공개 범위 |
| 신뢰와 안전 | Privacy Guard, 공유 범위, 사람의 최종 판단, 근거 중심 설명 |
| FAQ | 발표 중 나올 수 있는 주요 질문에 대한 보조 설명 |
| 부록 · StartFit 바로가기 | 실제 StartFit 랜딩페이지, MVP 진단 시작점, 대시보드 링크 |

## 코드 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 페이지 구조와 정적 콘텐츠 |
| `content.js` | 진단 과정 탭 데이터와 창업 유형 설명 데이터 |
| `icons.js` | SVG 아이콘과 텍스트 기반 아이콘 매핑 |
| `script.js` | 메뉴, 탭, 진단 흐름도, 이미지 확대, 해시 링크 상호작용 |
| `styles.css` | 디자인 토큰, 레이아웃, 반응형 스타일 |
| `MAINTENANCE.md` | 추후 수정 관리 가이드 |
| `.vercelignore` | 배포 시 임시 검증 파일과 로컬 의존성 제외 |

## 로컬 확인

정적 사이트라서 별도 빌드 없이 확인할 수 있습니다.

```bash
npx serve .
```

또는 원하는 정적 서버로 프로젝트 루트를 열면 됩니다.

## 배포

Vercel 정적 배포에 맞춰 `vercel.json`을 포함했습니다.

```bash
npx vercel --prod
```

## 검증 기준

- 데스크톱/모바일 렌더링 확인
- 메뉴 열기/닫기 확인
- 진단 과정 5개 큰 탭과 28개 화면 이동 확인
- 창업 유형 설명 5개 카드 확인
- 이미지 확대 모달 확인
- 대시보드 탭과 AI 세부 탭 이미지 전환 확인
- 기술 영역 정보 제공 탭과 RAG/스코어링 해시 링크 확인
- 깨진 이미지 없음
- 콘솔 오류 없음
- 모바일 가로 넘침 없음
